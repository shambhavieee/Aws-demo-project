A complete, full-stack web application featuring a responsive user interface and a secure, high-performance REST API. This project is fully containerized and designed for seamless cloud deployment.

## 🚀 Tech Stack

**Frontend:**
* React.js
* TypeScript
* Tailwind CSS

**Backend:**
* Golang
* SQL Database
* JWT (JSON Web Tokens) for secure authentication

**DevOps & Deployment:**
* Docker & Docker Compose
* AWS EC2

## 🛠️ Prerequisites

To run this project locally, you will need to have the following installed:
* [Docker](https://www.docker.com/get-started)
* [Docker Compose](https://docs.docker.com/compose/install/)
* Git
